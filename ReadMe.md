# All-The-Beans-Project

## Clone the Repository  

1.  Open a terminal or command prompt.  
2.  Clone the project repository:  
    ```bash
        git clone https://github.com/PanagiotisTsel/BeansTech.git
    ```
3.  Navigate to the project folder:  
    ```bash
    cd AllTheBeans.UI
    ```

## How to run
1.  Run and build the app
    ```bash
    dotnet run build
    ```
2.  Open the URL shown in the terminal (https://localhost:5001) in your browser
